Markdown 语法介绍

什么是 Markdown

Markdown 是一种轻量级的标记语言，常用于格式化文本，例如在博客、社交媒体、说明文档等地方编写和展示文本内容。
与 Word 不同，Markdown 是纯文本可移植的，几乎可以使用任何应用程序打开包含 Markdown 格式的文本文件。并且，Markdown 语法简单易学，通过 Markdown 应用，还可以很轻松地将 Markdown 文档转为 HTML 等格式。

Markdown 优点如下：

  1. 专注于文字内容；
  2. 纯文本，易读易写，可以方便地纳入版本控制；
  3. 语法简单，没有什么学习成本，能在轻松码字的同时做出美观大方的排版。

希望在线尝试 Markdown 语法，可以在这个在线网站尝试。本地也可以使用记事本，或者 Markdown 编辑器（如 Typora）来编辑。
Typora 中使用大纲模式可以在标准状态下进行预览，使用源代码模式可以用 Markdown 语法来编写文档：

Markdown 语法简介

一些常见的 markdown 语法如下：

  • 标题：使用#符号表示标题，使用一个#表示一级标题，两个#表示二级标题，以此类推，最多可以支持到六级标题。

  • 文本样式：使用*、_和~~包裹的文字会被添加样式，如**加粗**、*倾斜*、~~删除~~。

  • 列表：使用-或+或*+空格+文字可以创建无序列表，使用数字.+空格+文字可以创建有序列表。

  • 图片和链接：使用![图片名](图片地址)的语法可以插入图片，使用[链接名](链接地址)的语法可以插入超链接地址。

  • 代码和代码块：`code`的方式可以插入代码，使用连续三个`可以创建一个代码块包裹代码。

  • …更加详细的语法可以参见教程

Hexo 简介

Hexo 是一个基于 Node.js 的静态网站生成器，可以通过 Markdown 或者其他渲染引擎解析文章，快速生成静态网站，适用于个人博客、技术文档、项目文档等场景。
